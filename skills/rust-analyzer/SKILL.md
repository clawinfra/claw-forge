# rust-analyzer LSP Skill

## Description
Language server integration for code intelligence, diagnostics, and navigation.

## Capabilities
- Go-to-definition, find references, hover info
- Real-time diagnostics and error detection
- Code completion and signature help
- Symbol search and workspace navigation
- Code actions and quick fixes
- Formatting and refactoring

## Usage
This skill is auto-activated when working with relevant file types.
The language server runs as a background process and communicates via LSP protocol.

## Configuration
No configuration required. The skill auto-detects project settings from standard config files.
