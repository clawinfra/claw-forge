Honest Roadmap

Things we're building

We'll tell you what's missing. Honesty builds better software.

🔌 MCP Plugin Support

Connect claw-forge agents to any MCP server as a data source or tool. AutoForge has MCP support — we're building ours with a cleaner interface.

🌐 Multi-machine Agents

Distribute agent waves across multiple machines. The pool manager handles routing — we need the distributed work queue layer.

☁️ Cloud State Backend

Currently SQLite — great for local development. We're adding PostgreSQL and a hosted cloud option for team use.

🧩 VS Code Extension

Embed the Kanban UI in VS Code. See agent progress without leaving your editor.
